# Boletim Escolar

## Objetivo

Cadastrar os alunos, as matérias e fazer a soma de cada nota dos alunos nas matérias que vão ser lidas, depois da soma será tirado a média das notas e apresentar se os alunos foram aprovados ou reprovados.

## Principais funcionalidades previstas

- Somar
- Tirar a média
- Apresentar se o aluno foi aprovado ou reprovado
